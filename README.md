# Project-Deploying-a-Scalable-AWS-Architecture-with-Infrastructure-as-Code
Design, implement, and test a scalable, cloud-based architecture on AWS. The project requires using multiple AWS services—including EC2, VPC, RDS, S3, Lambda, and CloudFormation—alongside Terraform for Infrastructure as Code (IaC).Interact with AWS through the Management Console, CLI, and Python Boto3.
